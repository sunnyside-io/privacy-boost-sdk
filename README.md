# PrivacyBoost SDK

Privacy protocol SDK for Ethereum — available as TypeScript/React packages and a CLI.

## Installation

### TypeScript SDK

```bash
npm install @sunnyside-io/privacy-boost
```

### React Hooks

```bash
npm install @sunnyside-io/privacy-boost-react
```

### CLI
